System Overview

Welcome to S.Y.L.V.I.S., a highly optimized, futuristic AI desktop assistant tailored specifically for Windows. Unlike generic chatbots, Sylvis acts as an autonomous local operator capable of interacting directly with your operating system to execute shell scripts, open folders, adjust settings, and carry out tasks on your behalf.

⚠️ SECURITY WARNING: By default, Sylvis operates behind an interactive Firewall Security Approval Modal. Every time Sylvis generates a system action (like running a terminal command), it will ask for your clearance. You can toggle Auto-Approve to let her run tasks completely hands-free, but only do so when you trust the agent's context.

Getting Started

To begin utilizing Sylvis, follow these core steps:

  1. Make sure your background connection to the Claude server is active (the app launches this automatically).
  2. Double-click the glowing visualizer orb to toggle the **Command Console**.
  3. Open **Settings** (⚙ icon) to select your AI brain provider and configure your text-to-speech keys.
  4. Press the **Scroll Lock** or **Pause/Break** key anywhere on your PC to trigger the voice recorder, speak your request, and release!

Obtaining Free API Keys

Sylvis supports premium cloud brains and realistic voice synthesizers. You can sign up and acquire access keys for these services completely for free.

🧠 NVIDIA NIM (Free Claude/Nemotron Brain)

NVIDIA offers free developer API keys to access state-of-the-art models (like Claude 3.5 Sonnet) hosted on high-speed servers.

  • Go to build.nvidia.com.
  • Sign up for a free NVIDIA Developer Account.
  • Click **Get API Key** and copy the generated key.
  • Where to add: Open your web browser to the local admin page: http://localhost:8082/admin and paste the key into the API configuration box.

🗣️ ElevenLabs (Realistic Voices)

Provides highly natural voice synthesis. The free tier gives you access to a rich directory of voices.

  • Sign up at elevenlabs.io.
  • Go to **Profile Settings** (bottom-left) and reveal/copy your **API Key**.
  • Explore the Voice Library, find a voice you like, add it to your voices, and copy its **Voice ID**.
  • Where to add: Click the **SETTINGS** button inside the Sylvis terminal panel, select **ElevenLabs** as the TTS Provider, and enter your API Key and Voice ID.

🎭 Typecast (Anime TTS)

Specializes in character and anime voice acting. Perfect for giving Sylvis a custom character personality.

  • Sign up at typecast.ai.
  • Go to the Developer/API Dashboard and copy your **API Key**.
  • Select a voice model (e.g. standard character IDs) and retrieve its ID.
  • Where to add: Click **SETTINGS** in the Sylvis terminal, choose **Typecast**, and fill in the API Key and Voice Actor ID.

Feature Breakdown

🎙️ System-Wide Hotkey Waking

You can call Sylvis even while full-screen in games or other applications. Press **Scroll Lock** or **Pause/Break** on your keyboard to instantly wake the assistant. The visualizer orb will shift to listening mode.

🛡️ Interactive Security Firewall

When Sylvis wants to run an action, a popup window shows the exact script or command. You have full control to click **APPROVE** or **ABORT** before any terminal action occurs, protecting your operating system.

🎛️ Dynamic Visualizer Widget

The compact desktop widget is 100% borderless and transparent. It shifts animations depending on the engine state:

  • Breathing Cyan: Idle
  • Green Pulse: Listening
  • Violet Spinners: Processing
  • Magenta Ripple: Speaking

⚙️ System Tray Minimizing

Right-click the desktop widget and select **Minimize to Tray** to clean up your workspace. Sylvis will hide in your taskbar icons. Pressing the hotkey will automatically bring her back to the desktop.

100% Offline AI Brain

Sylvis has a secondary local brain that runs completely offline on your local computer's hardware, with no API keys, networks, or cloud endpoints required.

How It Works

When you switch the active AI Brain in Settings to **"Local Offline AI"**, Sylvis utilizes your CPU/RAM to load a highly optimized local model:

Qwen/Qwen2.5-0.5B-Instruct (Lazy-Loaded)

What to Expect

  • First Run Setup: On the first launch, the local model weights will be downloaded to your cache folder. This requires an active internet connection once.
  • Conversational Only: The local model functions as a chat companion (with a friendly personality). Because it runs on a compact local CPU model, it **cannot** run system shell commands or modify your OS.
  • Zero Latency Chat: After first-run setup, it operates 100% offline. You can disconnect your network completely and still talk to Sylvis!
